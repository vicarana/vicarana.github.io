# vicarana.github.io

Victor Arana's personal portfolio site — served at **https://vicarana.github.io**

Single-file, no build step. Edit `index.html` and push.

## Local preview
```bash
open index.html
```

## Deploy
```bash
git add -A && git commit -m "update" && git push -u origin main
```
Pages auto-rebuilds in ~60 seconds.
